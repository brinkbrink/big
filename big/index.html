Welcome

Welcome to my BIG project for IT 161, a website displaying some examples I've created with web development fundamentals. This includes research on three topics integral to powerful web development; accessibility, social media optimization (SMO), and HTML forms. Within these webpages I've also shown my understanding of HTML and CSS basics by customizing skeleton code given in class. This involved desiging the look of the pages with CSS, creating a php contact page, using Lightbox II to generate an interactive photo gallery, making lists and links, and the responsive embedding of YouTube videos, a Google calendar, and a Google map.

Continuous education is a personal goal of mine and a neccessity to moving within the web development field. Some of the ways I plan to obtain this goal:

  • Continuing to build my portfolio and new projects
  • Keeping up with relevant news with Slash Dot, ARS Technica, Tech Crunch, Smashing Magazine, and Hacker News
  • Pursue additional certifications that are valuable within the field
  • Learning new languages and updates to ones already mastered
  • Interacting with fellow developers by joining coding clubs and via forums like Github and Discord
